###################
🚨 Aplikasi Penyaluran Bantuan BPBD Sungai Penuh
###################


Aplikasi ini digunakan untuk membantu proses pendataan, pengelolaan, dan penyaluran bantuan pada Badan Penanggulangan Bencana Daerah (BPBD) Kota Sungai Penuh.

Sistem ini bertujuan untuk meningkatkan efisiensi, transparansi, dan akurasi dalam distribusi bantuan kepada masyarakat terdampak bencana.



*******************
Teknologi yang Digunakan
*******************

Backend: PHP / CodeIgniter (sesuaikan)
Frontend: HTML, CSS, JavaScript
Database: MySQL
Library:
DataTables
Bootstrap
